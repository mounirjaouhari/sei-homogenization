# `data/` — Numerical data

| File | Content |
|---|---|
| `constituents_properties.csv` | SEI constituent properties (LiF, Li₂CO₃, Li₂O, organic, …): `E, ν, K, G, ρ, D_Li, K_Ic, g, β`, Materials-Project metadata |
| `effective_parameters.csv` | effective interface parameters per composition (legacy values, kept for the before/after audit) |
| `effective_parameters_corrected.csv` | regenerated by `pipeline/recompute_effective.py` with the dimensionally-correct conventions (base `η=ε/L`, volumetric `g_v`) |

The corrected CSV is the authoritative one; the legacy CSV is retained only to
document the corrections (see `docs/PROGRAMMES_AUDIT.md`,
`docs/FE_VALIDATION_REPORT.md`).
